Laboratorio 2 de Sistemas operativos 1/2025

- Para compilar el programa se debe ingresar en terminal el comando

make

- Para ejecutar el programa se debe ingresar un comando con la forma

./desafio2 -t 10 -M 10 -p 4

Este comando de ejemplo creara el token(-t) 10, numero maximo de
decrecimineto(-M) 10 y cantidad de procesos creados(-p) 4
